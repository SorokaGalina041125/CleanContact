# 📞 CleanContact - Очистка адресной книги

Учебный проект по обработке и нормализации данных адресной книги с использованием регулярных выражений.

## 🎯 Задачи проекта

Необходимо было привести "грязные" данные адресной книги к единому формату:

1.Привести ФИО к правильному формату - разделить на фамилию, имя и отчество
2.Нормализовать номера телефонов - единый формат +7(999)999-99-99 (с добавочными номерами)
3.Удалить дубликаты - объединить повторяющиеся записи об одном человеке

## ✅ Что реализовано

- 📇 Разделение ФИО на отдельные поля (фамилия, имя, отчество)
- 📱 Приведение телефонов к формату +7(999)999-99-99
- 🔍 Поиск и объединение дублирующихся записей
- 💾 Сохранение результата в новый CSV файл

## 🚀 Установка и запуск

1.Клонировать репозиторий

bash
git clone <https://github.com/SorokaGalina041125/CleanContact.git>
cd CleanContact

2.Создать и активировать виртуальное окружение

bash

Windows
python -m venv .venv
.venv\Scripts\activate

Mac/Linux
python3 -m venv .venv
source .venv/bin/activate

3.Установить зависимости

bash
pip install -r requirements.txt

4.Подготовить данные

Поместить исходный файл phonebook_raw.csv в папку data/

5.Запустить обработку

python main.py

6.Результат

Обработанные данные сохранятся в data/phonebook.csv

📊 Формат данных
Исходный файл должен быть в формате CSV с полями:

text
lastname,firstname,surname,organization,position,phone,email

Python 3
